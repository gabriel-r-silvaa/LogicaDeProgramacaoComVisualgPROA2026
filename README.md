## 💻 Exemplo - Exercício 1 FacCat (Troca de Variáveis)

```visualg
Algoritmo "Exercicio_01_FacCat"
Var
   a, b, aux: inteiro
Inicio
   a <- 10
   b <- 20
   aux <- a
   a <- b
   b <- aux
   escrevaln("A: ", a)
   escrevaln("B: ", b)
Fim
```

**Saída esperada**:

A: 20
B: 10

## 🚀 Como Executar

1. **Baixe o VisualG** → [sourceforge.net/projects/visualg30](https://sourceforge.net/projects/visualg30/)
2. **Extraia** o arquivo baixado (não precisa instalar)
3. **Abra os arquivos `.alg`** do repositório
4. **F9** para compilar e executar
5. Teste com diferentes entradas
